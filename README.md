# MyFlowerArt-Web
ASP.NET Core 8.0 &amp; SQL Server kullanılarak geliştirilen interaktif çiçek tasarım ve e-ticaret platformu. Entity Framework Core (Code-First), Cookie Authentication tabanlı güvenli admin paneli ve gerçek zamanlı sipariş takip sistemi içerir. Kullanıcıların kendi buketlerini tasarlayabildiği dinamik bir alışveriş deneyimi sunar.
